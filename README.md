 Mirt Ta'em (ምርት ጣዕም) - Culinary Excellence Platform

Mirt Ta'em is a full-stack web application designed to connect users with high-quality culinary service providers. This project was developed as part of my professional software development portfolio while studying Information Systems at Bahir Dar University.

 🚀 Technologies Used
Frontend: React.js with Vite
Styling: Material UI (MUI), Tailwind CSS, and Shadcn UI
Language: TypeScript / JavaScript
Backend: Node.js and Express 
Database: SQL 
Deployment: Vercel (Frontend)

 ✨ Key Features

Service Listings: Browsing culinary experts and catering services.
Dynamic UI: Fully responsive design with cinematic image enhancements and realistic retouching aesthetics.
Management Logic: Integrated systems for service tracking, inspired by the management software I developed for local business operations.
Secure Authentication: User and provider login powered by Supabase.

🛠️ Installation & Setup

1.  Clone the repository:
    ```bash
    git clone [https://github.com/seli-mesfin/mirt-taem-frontend.git](https://github.com/seli-mesfin/mirt-taem-frontend.git)
    ```

2.  Navigate to the project folder:
    ```bash
    cd mirt-team
    ```

3.  Install dependencies:
    ```bash
    npm install
    ```

4.  Run the development server:
    ```bash
    npm run dev
    ```
